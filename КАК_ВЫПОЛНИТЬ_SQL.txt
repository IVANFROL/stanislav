═══════════════════════════════════════════════════════════════
  ИНСТРУКЦИЯ: Как создать базу данных в SQL Server
═══════════════════════════════════════════════════════════════

❌ НЕ ИСПОЛЬЗУЙТЕ "Import Data-tier Application"!
✅ Нужно просто открыть и выполнить SQL скрипт

═══════════════════════════════════════════════════════════════
ШАГ 1: Скопируйте SQL файлы на Windows
═══════════════════════════════════════════════════════════════

Скопируйте файлы CreateDatabase.sql и InsertSampleData.sql 
на ваш Windows компьютер в папку, например:
C:\Users\ВашеИмя\Desktop\stanislav\Database\

═══════════════════════════════════════════════════════════════
ШАГ 2: Откройте SQL Server Management Studio (SSMS)
═══════════════════════════════════════════════════════════════

1. Запустите SSMS
2. Подключитесь к серверу:
   - Server name: localhost (или localhost\SQLEXPRESS для Express)
   - Authentication: Windows Authentication

═══════════════════════════════════════════════════════════════
ШАГ 3: Откройте SQL скрипт
═══════════════════════════════════════════════════════════════

1. В SSMS нажмите: File → Open → File (или Ctrl+O)
2. Найдите файл CreateDatabase.sql
3. Откройте его

═══════════════════════════════════════════════════════════════
ШАГ 4: Выполните скрипт
═══════════════════════════════════════════════════════════════

1. Убедитесь, что весь скрипт выделен (Ctrl+A)
2. Нажмите F5 или кнопку "Execute" (Выполнить)
3. Дождитесь сообщения: "База данных AutoPartsStore успешно создана!"

═══════════════════════════════════════════════════════════════
ШАГ 5: Добавьте тестовые данные (опционально)
═══════════════════════════════════════════════════════════════

1. Откройте файл InsertSampleData.sql (File → Open → File)
2. Нажмите F5 для выполнения
3. Дождитесь сообщения об успешном добавлении данных

═══════════════════════════════════════════════════════════════
ПРОВЕРКА: Убедитесь, что база создана
═══════════════════════════════════════════════════════════════

В левой панели SSMS (Object Explorer):
1. Нажмите правой кнопкой на "Databases"
2. Выберите "Refresh" (Обновить)
3. Должна появиться база данных "AutoPartsStore"
4. Разверните её → Tables
5. Должны быть видны все таблицы (Users, Products, Orders и т.д.)

═══════════════════════════════════════════════════════════════
ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
═══════════════════════════════════════════════════════════════

1. Убедитесь, что SQL Server запущен:
   - Откройте Services (services.msc)
   - Найдите "SQL Server (MSSQLSERVER)" или "SQL Server (SQLEXPRESS)"
   - Статус должен быть "Running"

2. Проверьте права доступа:
   - Используйте Windows Authentication
   - Или убедитесь, что у вашего SQL пользователя есть права на создание БД

3. Если ошибка с кодировкой:
   - В SSMS: Tools → Options → Environment → International Settings
   - Установите кодировку UTF-8

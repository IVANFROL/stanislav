# Система управления магазином автозапчастей

## Описание

Полнофункциональная система управления магазином автозапчастей с базой данных SQL Server и пользовательским интерфейсом на Windows Forms.

## Возможности

- ✅ Управление товарами (автозапчастями)
- ✅ Управление клиентами
- ✅ Управление поставщиками
- ✅ Создание и управление заказами
- ✅ Учет продаж
- ✅ Система авторизации с двумя ролями (Админ/Пользователь)
- ✅ Контроль остатков товаров
- ✅ Отчеты по низким остаткам

## Требования

- .NET 6.0 или выше
- SQL Server (Express или выше)
- Visual Studio 2022 или выше

## Установка

### 1. Создание базы данных

1. Откройте SQL Server Management Studio (SSMS)
2. Подключитесь к вашему серверу SQL Server
3. Откройте файл `Database/CreateDatabase.sql`
4. Выполните скрипт для создания базы данных и таблиц
5. (Опционально) Выполните `Database/InsertSampleData.sql` для добавления тестовых данных

### 2. Настройка строки подключения

Откройте файл `AutoPartsStore/Program.cs` и измените строку подключения:

```csharp
var connectionString = "Server=localhost;Database=AutoPartsStore;Integrated Security=True;TrustServerCertificate=True;";
```

Если используется SQL Server Authentication:
```csharp
var connectionString = "Server=localhost;Database=AutoPartsStore;User Id=sa;Password=your_password;TrustServerCertificate=True;";
```

### 3. Запуск приложения

1. Откройте проект в Visual Studio
2. Восстановите NuGet пакеты
3. Соберите проект (Build)
4. Запустите приложение (F5)

## Учетные записи по умолчанию

После создания базы данных доступны следующие учетные записи:

### Администратор
- **Логин:** `admin`
- **Пароль:** `admin123`
- **Права:** Полный доступ ко всем функциям системы

### Пользователь
- **Логин:** `user`
- **Пароль:** `user123`
- **Права:** Просмотр данных, создание заказов и продаж

## Структура базы данных

### Основные таблицы:
- **Users** - Пользователи системы
- **Categories** - Категории товаров
- **Suppliers** - Поставщики
- **Customers** - Клиенты
- **Products** - Товары (автозапчасти)
- **Orders** - Заказы
- **OrderItems** - Позиции заказов
- **Sales** - Продажи
- **SaleItems** - Позиции продаж
- **Deliveries** - Поставки
- **DeliveryItems** - Позиции поставок

## Права доступа

### Администратор может:
- Просматривать все данные
- Добавлять, редактировать и удалять товары
- Управлять клиентами и поставщиками
- Создавать заказы и продажи
- Управлять пользователями системы
- Просматривать все отчеты

### Обычный пользователь может:
- Просматривать товары, заказы, продажи, клиентов
- Создавать новые заказы
- Создавать новые продажи
- НЕ может редактировать/удалять товары
- НЕ может управлять пользователями

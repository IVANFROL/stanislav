  ВАЖНО: Скопируйте файлы CreateDatabase.sql и InsertSampleData.sql
         на ваш Windows компьютер!
═══════════════════════════════════════════════════════════════

СПОСОБ 1: Через общий доступ или облако
─────────────────────────────────────────
1. Загрузите файлы CreateDatabase.sql и InsertSampleData.sql 
   в облако (Google Drive, Dropbox, OneDrive)
2. Скачайте их на Windows компьютер
3. Сохраните в папку, например: C:\Users\ВашеИмя\Desktop\stanislav\Database\

СПОСОБ 2: Скопировать содержимое вручную
─────────────────────────────────────────
1. Откройте файл CreateDatabase.sql на Mac
2. Выделите весь текст (Ctrl+A или Cmd+A)
3. Скопируйте (Ctrl+C или Cmd+C)
4. На Windows откройте Блокнот (Notepad)
5. Вставьте текст (Ctrl+V)
6. Сохраните как CreateDatabase.sql (тип файла: All Files, кодировка: UTF-8)
7. Повторите для InsertSampleData.sql

СПОСОБ 3: Через флешку или сеть
─────────────────────────────────────────
1. Скопируйте папку Database на флешку
2. Перенесите на Windows компьютер

═══════════════════════════════════════════════════════════════
ЧТО ДЕЛАТЬ ДАЛЬШЕ НА WINDOWS:
═══════════════════════════════════════════════════════════════

1. Откройте SQL Server Management Studio (SSMS)

2. Подключитесь к серверу:
   - Server name: localhost (или localhost\SQLEXPRESS)
   - Authentication: Windows Authentication
   - Нажмите Connect

3. В SSMS: File → Open → File (или Ctrl+O)
   - Найдите CreateDatabase.sql
   - Откройте его

4. Нажмите F5 или кнопку Execute (Выполнить)
   - Должно появиться сообщение об успешном создании БД

5. (Опционально) Откройте InsertSampleData.sql и выполните его (F5)

6. Проверьте в Object Explorer:
   - Databases → AutoPartsStore → Tables
   - Должны быть видны все таблицы
